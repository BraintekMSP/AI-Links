# Security Policy

## Scope

This repository is documentation and template content, not a deployed runtime.

## Reporting

If you find:

- embedded secrets
- sensitive example data
- private internal references that should not be public
- unsafe destructive guidance

report it privately to the repository owner before opening a public issue.

## Content expectations

- no live credentials
- no client data
- no private network details
- no default guidance that encourages unsafe agent behavior
