# Branding

This directory is the repo-authored source lane for Anarchy-AI branding facts.

Current rule:

- future forks should change brand metadata, legal URLs, logos, and publish-ready brand materials here first
- setup/build/manifests should consume generated branding artifacts from this source instead of hard-coding brand literals repeatedly
- this directory is intentionally unscaffolded for company-specific variants right now

Subdirectories:

- `./assets/` holds source branding assets such as logos, icons, and future fork-specific artwork
- `./published-materials/` holds publish-ready brand surfaces such as handoff copy, packaging text, and future instruction additions
