# Contributing

## Contribution expectation

This repository is public for reference and starter use only.
Publication does not create any support obligation or maintenance commitment.
Pull requests may be ignored or closed without review.

## Goal

Contributions should improve clarity, safety, and reuse for AI-assisted software delivery.

## Preferred contribution areas

- prompt clarity
- repo guardrail patterns
- public-safe templates
- maturity models
- cross-repo contract examples
- subagent safety patterns

## Avoid by default

- private/internal examples
- company-specific secret handling details
- tooling assumptions that only work on one machine
- destructive cleanup guidance without strong safety framing

## Change style

- keep examples generic
- prefer short, practical docs over long theory dumps
- update the runbook or changelog when the repo shape changes materially
